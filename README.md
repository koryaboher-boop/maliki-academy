# أكاديمية الإمام مالك للفقه المالكي

منصة تعليمية متكاملة لتدريس الفقه المالكي على مذهب الإمام مالك بن أنس.

## الميزات

- **دروس تفاعلية** مع نص المتن والشرح المبسط والموسع
- **فيديوهات تعليمية** مع مزامنة الصوت والنص
- **اختبارات قصيرة** بعد كل درس
- **خرائط ذهنية** وجداول تلخيصية
- **مكتبة علمية** لكتب المذهب المالكي
- **دعم الوضع الليلي والنهاري**
- **تصميم إسلامي عصري** بخط Amiri و Cairo
- **دعم كامل للغة العربية (RTL)**

## التقنيات المستخدمة

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- shadcn/ui
- Framer Motion

## التثبيت

```bash
# تثبيت الحزم
bun install
# أو
npm install

# تشغيل خادم التطوير
bun run dev
# أو
npm run dev

# افتح المتصفح على http://localhost:3000
```

## الدروس المتوفرة

### الوحدة الأولى: مقدمة في علم الفقه ومذهب مالك
- تعريف الفقه ومكانته من الشريعة
- ترجمة الإمام مالك ومنهجه
- أصول المذهب المالكي
- نشأة المذهب المالكي وانتشاره

### الوحدة الثانية: كتاب الطهارة (متن الأخضري)
- مقدمة متن الأخضري
- أحكام المياه
- فرائض الوضوء وسننه
- نواقض الوضوء
- الغسل من الجنابة
- الحيض والنفاس

### الوحدة الثالثة: كتاب الصلاة (متن الأخضري)
- مواقيت الصلاة: المختار والضروري والقضاء
- شروط الصلاة وفرائضها: فقه الضرورة والأركان الأربعة عشر
- سنن الصلاة وفضائلها ومكروهاتها: فقه الخشوع والكمال
- الخشوع وصلاة العاجز وقضاء الفوائت: فقه القلوب والذمة
- سجود السهو: فقه الخطأ البشري وعلاج الوسوسة
